Landing page for the CAworks AI sales assistant product.

## Deploy via GitHub Pages

1. Go to repository **Settings** → **Pages**
2. Under **Source**, select branch `main` and folder `/ (root)`
3. Click **Save** — the page will be live at `https://progprogect.github.io/AI-Agents-CRM-LP/`
